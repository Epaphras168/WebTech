# Web Technology Assignment: Java Servlets

## Project Overview
This project contains two Java Servlet assignments focused on form handling, server-side logic, and HTTP redirection using **Jakarta EE 10** and **Apache Tomcat 10**.

- **Assignment 1:** A Login Servlet that validates password strength.
- **Assignment 2:** A Search Servlet that redirects user input to a Google search query.
