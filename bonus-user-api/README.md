# User Profile API

### Base URL: /api/users

- GET /api/users – Get all users
- GET /api/users/{userId} – Get user by ID
- POST /api/users – Create new user
- PUT /api/users/{userId} – Update user
- DELETE /api/users/{userId} – Delete user
- GET /api/users/search?username=&country=&minAge=&maxAge= – Search users
- PATCH /api/users/{userId}/status?active=true/false – Activate/deactivate user
